How to use Django

  1. First Install: "pip install django" in terminal

  2. Mkdir: Mkdir MIT_DataVis

  3. Startproject: "django-admin startproject AirForceInnovation26"

    4. This creates your directories and sets up the flow of connections.

  4. Startapp: "django-admin startapp repo"

    5. prefferably something like personal or repo like I have.

  5. Runserver: "python manage.py runserver"

    6. Make sure you have manage.py in your AirForceInnovation26 file after you do startproject.

  6. Website Loaded: Copy and Paste url: http://127.0.0.1:8000/

    7. You should have a django home page with a rocket congratulating you.
